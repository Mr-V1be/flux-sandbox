# Screenshots

Drop live-demo captures here as `01-volcano.png`, `02-reactor.png`, etc. Keep
each image at 1600×1000 or smaller so the repo stays lean.

Reference them in the root `README.md` like this:

```md
![Volcano scenario](docs/screenshots/01-volcano.png)
```

## Shot list

- `01-volcano.png` — `Scenarios → Volcano` after 20 s of running
- `02-reactor.png` — `Scenarios → Reactor` right after detonation
- `03-circuit.png` — `Scenarios → Circuit` with a spark mid-flight
- `04-mayhem.png` — `Scenarios → Mayhem` with heat overlay (`T`) enabled
- `05-portals.png` — `Scenarios → Portal Loop` with sand flowing

## Tips

- Use `F` to fit-view, then `Ctrl+Wheel` to zoom in on a highlight region.
- Pause (`Space`) for crisp static frames.
- Enable the heat overlay (`T`) for the thermal-gradient screenshots.
